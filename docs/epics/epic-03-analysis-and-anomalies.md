# Epic 03 - Analysis and Anomalies

## Goal
Generate timeline summaries and detect anomalies with explanations.

## Scope
- Timeline derivation (grouped by time window)
- Heuristic anomaly detector
- Confidence scoring
- Store anomalies in DB

## Acceptance Criteria
- Analysis results include a timeline summary.
- Anomalies returned with rule, explanation, and confidence.
- Results link back to source events when applicable.
